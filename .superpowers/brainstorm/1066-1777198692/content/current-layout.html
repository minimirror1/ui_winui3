문제점 요약

1. 그룹핑 없음
연결/프로토콜/앱설정이 하나의 평면 목록으로 나열됨
2. 가상 모드 위치 불일치
연결 Expander 밖에 있어 논리적 흐름이 끊김
3. PING 카드 과부하
5개 하위 설정이 하나의 SettingsCard에 비표준 StackPanel로 묶임 — SettingsExpander로 분리해야 함
4. 하드코딩 문자열 혼재
일부는 다국어 리소스, 일부는 한국어 하드코딩
5. 응답 타임아웃 위치
